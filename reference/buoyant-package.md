# buoyant: Deploy '\_server.yml' Compliant Applications to 'DigitalOcean'

Provides tools to deploy R web server applications that follow the
'\_server.yml' standard. This standard allows different R server
frameworks ('plumber2', 'fiery', etc.) to be deployed using a common
interface. The package supports deployment to 'DigitalOcean' and
includes validation tools to ensure '\_server.yml' files are correctly
formatted.
